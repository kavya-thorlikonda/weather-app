# 🌤️ Weather App

A simple and responsive weather application built using **HTML, CSS, and JavaScript**.
This app allows users to enter a city name and get **real-time weather information** such as temperature, humidity, and weather conditions.

The application fetches weather data using the OpenWeatherMap API.


## 🚀 Features

* Search weather by city name
* Displays current temperature
* Shows humidity level
* Weather description
* Weather emoji based on conditions
* Error handling for invalid city names
* Clean and responsive UI


##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API

##  How It Works

1. The user enters a **city name** in the input field.
2. The app sends a request to the **OpenWeatherMap API**.
3. The API returns weather data in JSON format.
4. JavaScript processes the data and displays:

   * City name
   * Temperature
   * Humidity
   * Weather description
   * Weather emoji


##  API Setup

1. Create an account at OpenWeatherMap.
2. Generate an API key.
3. Replace the API key in the JavaScript file.

Example:

const apiKey = "YOUR_API_KEY";


## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your browser
4. Enter a city name
5. Click **Get Weather**
